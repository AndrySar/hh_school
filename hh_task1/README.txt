Задание1. Полином
	Дано выражение, содержащее скобки, операции сложения, вычитания, 
умножения, возведения в константную степень и одну переменную, 
например: (x - 5)(2x^3 + x(x^2 - 9)).

Представьте это выражение в развёрнутом виде, например: 3x^4 - 15x^3 - 9x^2 + 45x

Внимание!!!
 Строка исходного выражения должна иметь следующий формат: каждый символ должен быть разделен 
 пробелом в том числе и скобки.
 Пример: (x - 5)(2x^3 + x(x^2 - 9)) <=> ( x - 5 ) * ( 2 * x ^ 3 + x * ( x ^ 2 - 9))

 Ввод/Вывод осуществляется через файлы input.txt/output.txt
